# n8n

## Descripción

n8n es una herramienta de automatización de flujos de trabajo que te permite conectar diferentes servicios y automatizar tareas.

## Requisitos

- Puerto 5678 (por defecto)
- Espacio en disco: ~500MB

## Instalación

```bash
./install.sh
# Selecciona opción 1
```

## Configuración

Edita el archivo `.env` para configurar:

- `N8N_PORT`: Puerto para acceder a la interfaz web
- `N8N_BASIC_AUTH_ACTIVE`: Activar autenticación básica
- `N8N_BASIC_AUTH_USER`: Usuario para autenticación
- `N8N_BASIC_AUTH_PASSWORD`: Contraseña para autenticación

## Acceso

Una vez iniciado, accede a: http://localhost:5678

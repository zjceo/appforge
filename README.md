# AppForge

Sistema de instalación y gestión de aplicaciones con Docker.

## Descripción

AppForge es una herramienta que simplifica la instalación y gestión de aplicaciones usando Docker Compose.

## Aplicaciones soportadas

- n8n
- NocoDB
- Evolution API
- Typebot
- Chatwoot
- Flowise
- MinIO
- MongoDB
- Redis
- RabbitMQ

## Instalación

```bash
./install.sh
```

## Documentación

Consulta la carpeta `docs/` para más información sobre la instalación y uso de cada aplicación.
